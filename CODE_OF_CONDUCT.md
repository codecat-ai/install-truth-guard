# Code of Conduct

## Our Pledge

We pledge to make participation in this project respectful and harassment-free
for everyone.

## Expected Behavior

- Be considerate and constructive.
- Focus criticism on ideas and implementation details.
- Respect maintainers' decisions about project scope.

## Unacceptable Behavior

Harassment, threats, discriminatory language, and sustained disruptive behavior
are not acceptable.

## Enforcement

Maintainers may remove content, close discussions, or restrict participation
when needed to protect the project community.
